# 📝 Dorm Curfew Tracker - User Guide

## Introduction
This program allows students to check in and check out of the dorm digitally and lets dorm managers monitor attendance in real time.

## How to Access
1. Open the program (`python -m src.main`)  
2. Choose your role:
   - Student
   - Dorm Manager

## Steps for Students
1. Enter your Student ID and passkey.
2. Press the **Check-in/out** button.
3. Check your status (On-time or Late) displayed by the program.

## Steps for Dorm Managers
1. Enter your Name and passkey.
2. Access the dashboard to see student attendance in real time.
3. Export records if needed.

## Troubleshooting / Notes
- Make sure your Python environment is set up correctly.
- Only students and dorm managers with accounts can log in.
- If something doesn’t work, contact your dorm staff.
